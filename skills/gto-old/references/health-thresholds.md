# Health Thresholds

This file contains thresholds for calculating system health scores.

## Score Ranges

- **Excellent**: 90-100
- **Good**: 75-89
- **Fair**: 60-74
- **Poor**: 40-59
- **Critical**: 0-39

## Component Weights

- Error count: 40%
- Unfinished business: 25%
- Code markers: 15%
- Test coverage: 10%
- Documentation: 10%

## Penalty Values

- Critical error: -20 points
- Major error: -10 points
- Minor error: -5 points
- TODO marker: -2 points
- FIXME marker: -5 points
