# Error Patterns

This file contains patterns for detecting errors in transcripts.

## Error Indicators

- "error:", "Error:", "ERROR:"
- "failed", "Failed", "FAILED"
- "exception", "Exception", "EXCEPTION"
- "traceback", "Traceback", "TRACEBACK"

## Error Context Patterns

- File paths with line numbers (e.g., `file.py:123`)
- Function names with error indicators
- Stack trace patterns
