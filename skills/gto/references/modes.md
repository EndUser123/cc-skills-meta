# GTO Modes

## full (default)

Runs deterministic detectors, dispatches domain analyzer agent, merges results.

Best for: Regular gap analysis, comprehensive reviews.

## quick

Runs deterministic detectors only. No agent dispatch.

Best for: Fast checks, CI integration, pre-commit hooks.

## agent-only

Skips deterministic detectors. Only runs agent analysis.

Best for: Deep dives on specific areas, supplementing deterministic results.
