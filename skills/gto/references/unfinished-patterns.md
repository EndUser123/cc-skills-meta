# Unfinished Business Patterns

This file contains patterns for detecting unfinished work.

## Open Tasks Indicators

- "TODO:", "FIXME:", "HACK:"
- "[ ]" checkbox markers
- "incomplete", "pending", "deferred"

## Open Questions Indicators

- Question marks at turn end
- "?" in user messages
- Uncertainty phrases: "not sure", "maybe", "need to check"

## Deferred Items Indicators

- "later", "defer", "postpone"
- "for now", "for the moment"
- "skip", "come back to"
